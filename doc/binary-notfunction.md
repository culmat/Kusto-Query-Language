# binary_not()

Returns a bitwise negation of the input value.

## Syntax

`binary_not(`*value*`)`

## Parameters

| Name | Type | Required | Description |
|--|--|--|--|
| *value* | long | &check; | The value to negate. |

## Returns

Returns logical NOT operation on a number: value.
