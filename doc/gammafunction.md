# gamma()

Computes [gamma function](https://en.wikipedia.org/wiki/Gamma_function)

## Syntax

`gamma(`*x*`)`

## Arguments

* *x*: Parameter for the gamma function

## Returns

* Gamma function of x.

## See also

For computing log-gamma function, see [loggamma()](loggammafunction.md).
