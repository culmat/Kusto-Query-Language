# exp2()

The base-2 exponential function of x, which is 2 raised to the power x: 2^x.  

## Syntax

`exp2(`*x*`)`

## Parameters

| Name | Type | Required | Description |
|--|--|--|--|
| *x* | real | &check;| The value of the exponent. |

## Returns

The exponential value of x.
