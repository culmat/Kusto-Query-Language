# tan()

Returns the tangent value of the specified number.

## Syntax

`tan(`*x*`)`

## Parameters

| Name | Type | Required | Description |
|--|--|--|--|
| *x* | real | &check; | The number for which to calculate the tangent. |

## Returns

The result of `tan(`*x*`)`
